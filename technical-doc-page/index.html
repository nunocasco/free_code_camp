Instalação

A instalação é fácil, e, hoje em dia, muitas distribuições Linux e UNIX (Apple, por exemplo, já trazem Python instalado. Até alguns Windows, como os HP, já vêm com Python instalado de fábrica!

Se precisas de instalar o Python no teu computador, podes encontrar ajuda neste guia para iniciantes

Aprendizagem

Antes de começares, dá uma vista de olhos na versão (IDE), que melhor se adequa às tuas configurações de hardware/software. Prossegue com a instalação!

Vamos experimentar?

Parabéns por teres chegado aqui, vai começar a diversão!

Por esta altura já deves ter instalado o Python e um editor de texto.

Abre o editor de texto e vamos começar!

Usa a função "print" para o resultado que pretendes mostrar, por exemplo:

print("Olá, mundo!")

Experimenta imprimir uma palavra ou frase que quiras, qualquer uma!

print("o que tu quiseres dentro de aspas")

É engraçado, não é? Mas também podes executar matemática! Experimenta:

print(2+3)

5? E que tal um bocadinho mais avançado?

x = 1 print(x+1)

Mas atenção que nem tudo corre bem! Há coisas que não se podem fazer...

print(1 + "Olá, mundo!")

Agora, estuda, pesquisa e diverte-te!

print("Parabéns, ...) "
